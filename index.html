Política de Privacidad

Última actualización: 16 de junio de 2026

Esta Política de Privacidad explica qué información procesa Flux Bot ("el Bot") cuando lo usas en Discord, y qué hacemos (y no hacemos) con ella.

Resumen rápido: Flux Bot no almacena de forma permanente datos personales de los usuarios. Toda la información se procesa en memoria durante la ejecución del comando y se descarta inmediatamente después.

1. Información que procesamos

Para funcionar, el Bot necesita leer cierta información mínima y temporal:

DatoOrigenUso
Enlace de TikTok / Instagram / YouTubeMensaje del usuarioObtener metadatos públicos del video (vistas, likes, duración, calidad, etc.)
ID y nombre de usuario de DiscordDiscord APIMostrar el remitente en la tarjeta de "self-promo" (mención o nombre visible)
Archivo de videoPlataforma de origenComprimirlo temporalmente para adjuntarlo en Discord, si supera el límite de subida

2. Qué NO almacenamos

  • No guardamos un historial de los enlaces o videos que analizas.
  • No guardamos tu ID de Discord, nombre de usuario ni ningún dato personal en una base de datos.
  • Los archivos de video descargados y/o comprimidos se escriben temporalmente en disco solo durante el procesamiento de un comando, y se eliminan automáticamente al finalizar (éxito o error).
  • No vendemos, compartimos ni transferimos ninguna información a terceros con fines publicitarios.

3. Servicios de terceros

El Bot se conecta a las APIs/páginas públicas de TikTok, Instagram y YouTube únicamente para obtener la información del enlace que proporcionas. El uso de estas plataformas está sujeto a sus propias políticas de privacidad, sobre las cuales no tenemos control:

  • Política de Privacidad de TikTok
  • Política de Privacidad de Instagram
  • Política de Privacidad de Google / YouTube

4. Discord

El Bot opera dentro de Discord y utiliza la API oficial de Discord para leer mensajes, responder y, en self-promo, eliminar el mensaje original del usuario cuando aplica. El uso de Discord está sujeto a su propia Política de Privacidad.

5. Retención de datos

Como no almacenamos datos personales de forma persistente, no existe un período de retención que gestionar: la información se descarta al completar cada solicitud.

6. Tus derechos

Dado que no mantenemos una base de datos de usuarios, no es necesario solicitar la eliminación de datos personales almacenados. Si tienes dudas sobre cómo el Bot procesa información en un caso específico, puedes contactarnos.

7. Cambios a esta Política

Podemos actualizar esta Política de Privacidad si cambian las funciones del Bot. Recomendamos revisar esta página periódicamente.

8. Contacto

Para preguntas sobre esta Política, puedes contactarnos a través de TikTok @_tagashy.tt.
